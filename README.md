# الحيَـاة — Hayat Travel & Tourism ✈️

إعادة تصميم كاملة لموقع [hayat.iq](https://hayat.iq) — شركة الحياة للسياحة والسفر، بغداد، منذ 2019.

A complete redesign of the Hayat Travel & Tourism website — built strictly on the company's corporate identity guidelines.

---

## 🧭 الفكرة التصميمية — Design Concept: «الرحلة»

الموقع نفسه **رحلة طيران**: من لحظة دخول المستخدم (بوابة الصعود / Preloader) وحتى وصوله للفوتر، تتغير سماء الصفحة بنفس تسلسل ألوان الهوية البصرية:

| المرحلة | القسم | اللون (من دليل الهوية) |
|---------|-------|------------------------|
| 🌅 الفجر | Hero | Light Lavender `#F5F8FF` → Sunset `#FFB4A4` |
| 🌇 الغسق | عن الحياة + الخدمات | Purple Sky `#250353` |
| 🌌 الليل | الوجهات → الفوتر | Dark Night `#0F0526` + نجوم متلألئة |

- **طائرة مرافقة**: شريط تقدم جانبي بشكل مسار رحلة (BGD → ✦) يتحرك مع التمرير
- **بطاقة صعود (Boarding Pass)** مستنسخة من دليل الهوية (صفحة 99) كعنصر تفاعلي
- **بطاقات الوجهات**: رسومات SVG يدوية بأسلوب بوسترات البراند (معالم بسيلويت فوق سماء غروب بنفسجية) — بدون أي صور خارجية

## 🎨 الالتزام بالهوية — Brand Compliance

- **الألوان**: الأربعة الأساسية + التدرجات (Sunset → Orchid → Purple) كما في صفحات Gradients & Colors Hierarchy
- **الشعار**: إعادة رسم الرمز الدائري (الطائرة/حرف H) كـ SVG + الاسم بالعربية والإنجليزية
- **الأيقونات**: خطوط بنهايات مدوّرة (Rounded Ends) كما في Iconography Guides
- **نبرة الصوت**: ودّية، ملهمة، بسيطة، متفائلة — العبارات من قسم Verbal Expressions
- **المحتوى**: الرؤية، الرسالة، القيم، شرائح الجمهور، وتطور العلامة (2019 → اليوم) من البروفايل

## ⚙️ التقنية — Tech

- **HTML5 + CSS3 + Vanilla JS** — صفر مكتبات خارجية (لا jQuery ولا GSAP)
- **ثنائي اللغة**: عربي RTL (افتراضي) ⇄ إنجليزي LTR بزر واحد، مع حفظ التفضيل
- **Responsive بالكامل**: موبايل / تابلت / لابتوب / شاشات كبيرة (CSS Grid + Logical Properties)
- **أنميشن**: IntersectionObserver + requestAnimationFrame + Canvas (نجوم) + SVG path animation (الطائرة تتبع المسار)
- **إتاحة**: `prefers-reduced-motion`، ARIA labels، تباين ألوان مدروس
- **أداء**: بدون صور نقطية إطلاقاً — كل الرسومات SVG/CSS، تحميل شبه لحظي

## 🚀 التشغيل — Run

```

```
